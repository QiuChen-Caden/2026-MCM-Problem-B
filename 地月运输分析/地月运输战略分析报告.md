# 战略级地月运输架构分析报告：当代重型火箭性能、成本口径与发射能力上限评估

在当代深空探索与地月经济圈建设的大背景下，运载火箭的运输效能是决定工程任务工期、成本与可行性的核心瓶颈。随着地月空间物流需求的指数级增长，从传统的消耗性发射模式向可重复使用、在轨补加模式的转变，不仅改变了载荷交付的物理逻辑，更重塑了发射场管理与全球航天供应链的运作方式。本报告旨在针对地月运输任务中的关键子任务，即载荷能力、成本结构与发射上限进行深度解构，为后续任务规划提供详实的技术支撑与模型参数。

## 运载能力与“月球交付点”的定义

在地月运输模型中，明确“交付点”是建立有效运载系数的前提。根据不同的任务需求，交付点通常分为近地轨道（LEO）、地月转移轨道（TLI）、月球轨道（如NRHO或LLO）以及月球表面。

### 月球交付点的层级与能量需求分析

由于动能增量（$\Delta v$）在空间操作中的决定性作用，同样的火箭在不同交付点的运载能力呈现剧烈波动。以当前最先进的可重复使用运载系统Starship为例，其设计的核心在于通过在轨补加（On-orbit Refilling）来打破齐奥尔科夫斯基方程的限制 。

|**交付点定义**|**典型任务描述**|**对应 Δv (从LEO出发)**|**载荷缩减系数（参考值）**|
|---|---|---|---|
|**LEO (近地轨道)**|轨道组装、推进剂加注|0 km/s (基准)|1.0|
|**TLI (地月转移轨道)**|航天器摆脱地球引力转向月球|~3.1–3.2 km/s|0.25–0.30|
|**NRHO (近直线晕轨道)**|月球门户（Gateway）对接|~3.6–3.8 km/s|0.15–0.20|
|**Lunar Surface (月面)**|软着陆、基地建设|~5.8–6.3 km/s|0.05–0.10|

对于题目中提到的“100–150吨到月球”的未来假设，必须明确其指的是“交付到月球表面”。在不具备在轨补加的情况下，当代没有任何单次发射任务能达到此量级。即便如史上最强的SLS Block 2，其交付到TLI的能力也仅为46吨以上，若要软着陆到月面，载荷将进一步缩减至个位数 。因此，100吨级的月面交付前提是实现在轨多次推进剂补给。

### 当代重型运力参考：从SLS到Falcon Heavy

为解释未来100-150吨参数的合理性，必须建立在当代真实数据之上。当前处于服役状态或即将服役的重型运载工具主要包括NASA的太空发射系统（SLS）、SpaceX的猎鹰重型（Falcon Heavy）以及正在测试中的星舰（Starship）。

|**火箭型号**|**配置模式**|**LEO运力 (t)**|**TLI运力 (t)**|**月面交付潜力 (t)**|
|---|---|---|---|---|
|**SLS Block 1**|全消耗型|95|27|< 5|
|**SLS Block 1B**|全消耗型|105|38–42|~8|
|**SLS Block 2**|全消耗型|130|46+|~10|
|**Falcon Heavy**|部分回收|63.8|~15|~2–3|
|**Starship Block 1**|完全回收|15 (测试期)|N/A|N/A|
|**Starship (最终目标)**|完全回收|100–150|N/A (需在轨补加)|100 (需加注)|

SLS通过牺牲回收性换取极高的深空投送能力，其Block 1配置已在Artemis I任务中验证了27吨到TLI的能力 。相比之下，Falcon Heavy的商业价格优势极其显著，虽然单次月面投送质量较小，但其每千克载荷的经济性为建立长期月球基地提供了更为可行的小步快跑方案 。

## 发射成本口径与经济性建模

发射成本的计算需区分固定成本与变动成本。固定成本包括研发摊销、地面保障设施维护及基本运营费用；变动成本则主要由推进剂、消耗性部件（如二级火箭）、回收整修费用及任务专属整合费用组成。

### 当代成本基准数据

根据2023至2025年的行业数据，不同运载系统的成本口径存在量级差异。

|**火箭型号**|**单次发射标称价格**|**载荷能力 (LEO)**|**单位质量成本 ($/kg - LEO)**|**月面交付单位成本 ($/kg)**|
|---|---|---|---|---|
|**Falcon 9**|$6,700万|22.8 t|~$2,940|N/A|
|**Falcon Heavy**|$9,700万|63.8 t|~$1,520|~$10,800|
|**SLS / Orion**|$41亿|95 t|~$43,000|~$150,000+|
|**Starship (目标)**|$200万–$1,000万|100 t|~$20–$100|~$1,000 (含补加)|

SLS的成本模型中，由于其使用了大量的航天飞机遗留硬件（如RS-25发动机和固体助推器）且不具备回收能力，单次发射成本高达25亿至41亿美元 。这种高昂的成本口径决定了其只能用于极少数关键性的载人任务。而SpaceX的猎鹰系列通过第一级助推器的回收，单次发射可节省约1500万美元的成本，从而将单位质量成本压低至传统模式的几十分之一 。

### 单位吨变动成本的非线性特征

在模型中，单位吨变动成本并不是简单的线性关系。对于Starship这类完全可重复使用的系统，推进剂成本在总成本中的占比将从传统火箭的不到1%上升至50%以上。Starship加注一次约需1200吨推进剂，其中液氧（LOX）与液甲烷（LCH4）的配比约为3.8:1 。

随着发射频率的增加，地面处理周期的变动成本将成为决定$/t的关键。例如，SpaceX计划在星舰成熟后，将发射成本降至每千克20美元甚至更低 。但在目前的测试阶段，星舰单次消耗性发射的价格仍被估算在1亿美元左右 。

## 发射能力上限与吞吐量约束：致命的潜在瓶颈

“10个发射场”并不意味着发射能力可以无限叠加。在实际操作中，发射频率受到地理、环境、技术和法规的多重制约。

### 场址年最大发射次数分析

目前的环评报告（EIS）和FAA许可为核心发射场设定了明确的硬上限。这些上限不仅取决于发射架的周转速度，还受到噪音控制、空域占用和公众安全的法律限制。

|**发射场址**|**许可/拟定上限 (次/年)**|**关键限制因素**|
|---|---|---|
|**KSC LC-39A (Starship)**|44|日间/夜间配额（各22次）、空域冲突|
|**CCSFS (Starship)**|76|美国空军/国防部任务优先权、地面基建|
|**Starbase (Boca Chica)**|目前5次 (拟增加)|环境保护、社区道路封闭限制|

在LC-39A的分析中，每年44次发射是一个重要的模型参数。这意味着即便所有10个潜在场址均按此标准运作，全球年总吞吐量也将被限制在400-500次以内 。对于单次月面任务需要10-15次补加发射的架构来说，这意味着单一发射场每年只能支撑约3-4次完整的月球货运任务 。

### 环境与地面处理周期约束

发射能力上限的另一个变量是地面处理周期（Turnaround Time）。猎鹰9号目前已实现了最短21天的助推器周转记录，而星舰的目标是实现小时级的周转 。然而，现实中的制约因素更为复杂：

1. **范围管制与空域封闭**：星舰发射会对大西洋航线造成显著干扰。据测算，每次发射需封闭空域40分钟至2小时，影响多达23,000架次民航飞行，每年的经济延迟成本高达数亿美元 。
    
2. **天气约束**：避雷 commit 准则规定，发射场周围10海里内30分钟内不得有闪电活动。此外，高空风切变（Upper-level winds）是导致发射洗刷（Scrub）的最常见原因 。
    
3. **地面物流压力**：支持星舰单次发射需要约440辆推进剂卡车（包括270辆液氧车、80辆液氮车和90辆甲烷车）。年发射44次意味着每年需调度近2万辆重型卡车，这对当地公路基础设施是巨大的压力 。
    

## 交付效率系数：从LEO载荷到月面交付的转换

在地月运输模型中，我们引入“交付效率系数” ($k_{delivery}$)，用于计算为了在月球表面交付单位质量所需的LEO入轨质量。

### 轨道加注与质量占比

对于高比冲化学火箭（如Raptor发动机，真空$I_{sp} \approx 363-380s$），从LEO到月球表面的$\Delta v$预算通常在5.8到6.3 km/s之间 。

根据齐奥尔科夫斯基方程计算：

$$m_{initial} / m_{final} = e^{(\Delta v / I_{sp} g)}$$

对于$I_{sp} = 370s, \Delta v = 6.0 km/s$的情况，质量比约为5.23。这意味着如果不计入蒸发损耗，在轨道上每5.23吨的初始质量才能换取1吨的月面软着陆质量。如果考虑到星舰本身约100-150吨的干质量，这一系数会进一步恶化 。

### 补加次数与敏感性分析

模型参数显示，单次月球着陆任务所需的加油发射次数与单次补加量高度敏感。

|**补加效率 (吨/次)**|**所需加油次数 (典型100t任务)**|**对发射场吞吐量的压力**|
|---|---|---|
|50 (初期/非理想)|30+|极高（超过单一场址年产能）|
|100 (标称目标)|10–15|高（占用场址25%-33%产能）|
|150 (未来优化)|4–8|适中（效率最高点）|

如果单次加油运力从100吨下降到50吨，任务所需的总发射次数将从约12次激增至30次以上，这将直接导致任务工期翻倍，并由于轨道蒸发（Boil-off）导致推进剂损失进一步扩大 。

## 轨道推进剂稳定性与蒸发损耗

推进剂蒸发损耗是模型中最常被忽略但最致命的变量。低温推进剂（液氧/甲烷）在LEO轨道上面临剧烈的热载荷。

1. **热流密度**：在LEO轨道上，星舰平均面临1150 $W/m^2$ 的综合热输入 。
    
2. **蒸发率**：在无主动冷却的情况下，甲烷和液氧的蒸发损失每天可达77吨。如果不采取隔热措施，星舰在两周内就会流失全部燃料 。
    
3. **技术对策**：通过多层绝热材料（MLI）可将蒸发率降至1吨/天以下。高阶模型必须包含这一时间常数，以确定加油发射的频率下限 。
    

## 综合评估与战略结论

基于对当前重型运载系统及其限制条件的深度研究，我们可以为“B. 火箭运输”子任务提供以下核心模型参数建议：

### 核心参数建议表

|**参数项**|**推荐取值范围**|**解释说明**|
|---|---|---|
|**单次载荷（月面）**|100–150 吨|基于在轨补加架构，干质量约100t，有效载荷100t+|
|**单次发射成本**|$6,700万 – $1.5亿|取当代商业成熟价，不考虑未来理想化的极低成本|
|**单位吨成本 ($/t)**|$10,000 – $15,000 (月面)|包含补加发射的分摊成本，远低于SLS的$15万|
|**单场址吞吐上限**|44 – 76 次/年|基于FAA EIS审批红线，非技术上限而是法规上限|
|**交付效率系数**|12:1 – 15:1|即每向月面交付1吨载荷，需从地面发射12-15吨质量至LEO|

### 关键因果关系与风险推论

模型建设者应关注以下几个层面的联动影响：

- **工期影响**：由于单一发射场每年仅能支持约3次大规模月面任务（假设补加需求为15次/任务），工期的缩短必须依赖于多场址同步建设。如果10个发射场能够克服物流和法规限制同步运行，月面交付频率可提升至30次/年，但这需要解决全球级的甲烷和液氧供应链问题 。
    
- **成本敏感性**：加油次数是成本波动的最大推手。如果回收技术不成熟导致补加次数增加，月面交付成本将呈现非线性增长。
    
- **环境瓶颈**：公路运输能力和空域管制是目前商业模型中最弱的一环。推进剂的本地化生产（ASU/气分装置）必须作为基建参数进入模型，否则卡车吞吐量将锁死发射频率上限 。
    

本报告所提供的参数旨在平衡航天工程的物理极限与现实中的运营约束。在实际任务执行中，应根据Starship Block 3及更高版本的实际飞行数据，动态调整补加效率系数与蒸发损耗率，以优化整体地月物流链的成本效益比。
